# Guqin Tablature Composer Create Guqin music with the NLTabs (Numeric Linear Tablature) system. This creates Guqin music with a simple entry method. Guqin Notation traditionally has been a bit tricky to read, since it's based on cramming a bunch of playing instructions into a set of characters, rather than encoding the actual notes and rhythms. More information on my Guqin notation ideas can be found [on my blog post here](https://blog.nyl.io/guqin-part-3-tablature-old-and-new/).

## What am I looking at? **Tablature**: The generated tab (right side of the screenshot) has music notes, and playing positions in the NLTab structure. 7 lines to represent the 7 strings, with numbers on each line to represent where to press or touch on the string (hui positions). **Shorthand Input**: You can input using a shorthand I've developed, or using LilyPond notation. I also created a [Shorthand Glossary](http://nyl.io:7013/nltabs/qbSWqqt/) here for common Guqin techniques written in the shorthand. Take a look at the [help section](/help.html) for details. **LilyPond Input**: You can also input using LilyPond. LilyPond is more powerful, but a lot more coding involved. It's a tool that renders any music notation (piano, orchestral scores, old timey guitar stuff), and I've modified it by adding some custom functions to render Guqin-specific tablature. You can see a the [LilyPond Glossary](http://nyl.io:7013/nltabs/J1QPkkY/) here for various Guqin techniques noted in LilyPond.
